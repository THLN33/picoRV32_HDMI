# picoRV32_HDMI
Software part of System_picoRV32. Include Bootloader and App HDMI Demo

This repository contains the embedded C software designed to run on a RISC-V SoC based on the PicoRV32 processor.

The corresponding hardware/RTL implementation can be found here: [System_picoRV32](https://github.com/THLN33/System_picoRV32).

## 🛠️ Eclipse IDE Configuration
    Install the Eclipse Embedded CDT plugin.
    Import this project: File > Import > Existing Projects into Workspace.
    Ensure the "Build Settings" point to your RISC-V toolchain path.
